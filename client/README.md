# Build for development

```
npm run build
```

or

```
yarn build
```

# Build for production
```
npm run build:production
```

or

```
yarn build:production
```

# Watch
```
npm run watch
```

or

```
yarn watch
```

# Test
```
npm run test
```

or

```
yarn test
```

# Lint
```
npm run lint
```

or

```
yarn lint
```
